# Summary of Personal Projects

This repository collects several personal Seeed Studio XIAO / ESP32 projects and notes, including Arduino examples, hardware prototypes, sensor applications, and community project collections.

## Projects

| Project | Description |
| --- | --- |
| `Smart_Stair_Light_with_mmWave_Radar/` | A dual-radar smart stair light prototype based on XIAO ESP32-C6, LD2410C mmWave radar, and ESP-NOW. It supports upstairs/downstairs direction detection and flowing LED animations. |
| `Visual_Recognition_Alarm/` | A visual recognition alarm example using Grove Vision AI V2 and XIAO ESP32 boards, with LED, buzzer, or audio warning output. |
| `XIAO-ESP32C5-WiFi-and-HA-Demo/` | Wi-Fi signal scanning and Home Assistant integration examples for XIAO ESP32C5. |
| `ESP32-C6-Deep-Sleep-with-External-Wake-Up/` | An ESP32-C6 deep sleep and external wake-up example, focusing on using ext1 wake-up on ESP32-C6. |
| `IMU-ColorSync-LED-Controller/` | A small IMU and LED controller project for syncing LED colors with motion or orientation data. |
| `XIAO-Community-Projects-Weekly-Blog/` | A weekly collection of creative XIAO community projects from GitHub, YouTube, Instructables, and other platforms. |

## How to Use

Each subfolder is an independent project. Most projects include a README file, Arduino `.ino` example code, and sometimes hardware files or result files. Open the corresponding project folder to find hardware requirements, software dependencies, and upload instructions.

This repository will continue to be updated with more project records and examples.

## 中文说明

这个仓库整理了一些个人做过或正在整理中的 Seeed Studio XIAO / ESP32 相关项目，主要包含 Arduino 示例、硬件原型、传感器应用和社区项目资料。

### 项目列表

| 项目 | 简介 |
| --- | --- |
| `Smart_Stair_Light_with_mmWave_Radar/` | 基于 XIAO ESP32-C6、LD2410C mmWave 雷达和 ESP-NOW 的双雷达智能楼梯灯原型，支持上下楼方向判断和 LED 流水灯效果。 |
| `Visual_Recognition_Alarm/` | 使用 Grove Vision AI V2 和 XIAO ESP32 系列开发板实现的视觉识别报警示例，可通过 LED、蜂鸣器或音频模块进行提醒。 |
| `XIAO-ESP32C5-WiFi-and-HA-Demo/` | XIAO ESP32C5 的 Wi-Fi 信号扫描和 Home Assistant 集成示例。 |
| `ESP32-C6-Deep-Sleep-with-External-Wake-Up/` | ESP32-C6 深度睡眠与外部唤醒示例，重点说明 ESP32-C6 使用 ext1 外部唤醒的方式。 |
| `IMU-ColorSync-LED-Controller/` | IMU 与 LED 控制相关的小项目，用于根据姿态或运动信息同步 LED 颜色效果。 |
| `XIAO-Community-Projects-Weekly-Blog/` | XIAO 社区项目周报合集，整理 GitHub、YouTube、Instructables 等平台上的 XIAO 创意项目。 |

### 使用说明

每个子目录都是一个独立项目，通常会包含项目 README、Arduino `.ino` 示例代码，部分项目还包含硬件文件或结果文件。进入对应目录后，可以查看该项目的 README 获取硬件准备、软件依赖和烧录步骤。

这个仓库会持续补充新的项目记录和示例。
